## Practical 6: Rectangle Class
Problem Statement:
Define a Java class named Rectangle with width and height as data members. Provide a no-argument constructor and a parameterized constructor. Implement methods to calculate and return the area and perimeter.

## Practical 7: Employee Class
Problem Statement:
Create an Employee class with private data members for employee name and salary. Implement methods to read employee data from the user and display the employee details. Demonstrate object creation and method invocation.

## Practical 8: Point Class
Problem Statement:
Create a Point class representing a two-dimensional point. Implement a default constructor, a parameterized constructor, and a copy constructor. Include a method to display the point coordinates.

## Practical 9: Rectangle Comparison
Problem Statement:
Create two Rectangle objects with predefined dimensions. Display their width, height, area, and perimeter. Compare the areas of both rectangles and display which one has the larger area.
