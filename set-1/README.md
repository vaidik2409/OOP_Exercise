# SET-1 (Basics of JAVA)

## Compilation Guide
To compile all Java files in this set:
javac *.java

---

## Practical 1: Distance Converter
Problem Statement:
Develop a Java program that prompts the user to enter a distance in meters. The program should convert the distance into feet using the conversion factor (1 meter = 3.28084 feet) and display the result formatted to two decimal places.

---

## Practical 2: Cramer's Rule (Linear Equations)
Problem Statement:
Write a Java program to solve a system of two linear equations with two variables using Cramer's Rule. The program should accept coefficients from the user, calculate the determinant, and compute the values of x and y. Include proper handling when the determinant is zero.

---

## Practical 3: Vowel Checker
Problem Statement:
Write a Java program that prompts the user to enter a single character and determines whether it is a vowel or a consonant. The program should work for both uppercase and lowercase letters.

---

## Practical 4: BMI Calculator
Problem Statement:
Develop a Java application that calculates Body Mass Index (BMI). The program should accept weight in pounds and height in inches, convert them into kilograms and meters respectively, and calculate the BMI value.

---

## Practical 5: Area of Triangle
Problem Statement:
Write a Java program that takes the lengths of three sides of a triangle as input. Validate whether the sides can form a triangle and calculate the area using Heron's formula.
